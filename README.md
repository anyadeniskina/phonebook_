# Телефонный справочник

Десктопное приложение для хранения и управления контактами с графическим интерфейсом и хранением данных в реляционной СУБД.

## Стек
- C++
- Qt (UI)
- PostgreSQL (хранение данных)

## Функциональность
- CRUD-операции над контактами (создание, чтение, обновление, удаление)
- Графический интерфейс на Qt
- Хранение данных в PostgreSQL
Проект развивался итеративно:
1. `task1` — консольная версия с базовой логикой
2. `task2` — добавлен графический интерфейс на Qt
3. `main` (бывш. `task3`) — финальная версия с хранением данных в PostgreSQL
